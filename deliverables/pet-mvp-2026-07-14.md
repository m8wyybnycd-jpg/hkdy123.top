# AI记忆体桌面宠物 MVP 交付报告

**日期**: 2026-07-14 | **构建状态**: ✅ tsc --noEmit + vite build 全绿

## 概述

在云玩汇(hkdy123.top)网站中集成了一个AI记忆体桌面宠物系统，用户登录后可自动领养一只虚拟精灵宠物。宠物具备AI对话能力（接入讯飞MaaS混元7B）、上下文感知（根据当前浏览页面给出精准建议）、5级成长系统和长期记忆。

## 文件清单

### 数据库
- `migrations/schema-pet.sql` — 5张表：pets, pet_memories, pet_conversations, pet_growth_logs, pet_page_contexts

- `functions/lib/pet.ts` — 共享类型与工具函数（经验系统、等级计算、每日上限检查）

## 技术实现亮点

### Canvas 2D 程序化渲染
- 无需任何外部精灵图，宠物完全通过 Canvas 2D 绘制
- 5级视觉：蛋 → 幼崽 → 成长 → 伙伴 → 专家（皇冠+闪光）
- 7种状态：idle/waving/jumping/running/failed/waiting/review
- 每帧可配置fps，支持眨眼动画、腮红、耳朵摆动

### SSE 流式对话
- 代理讯飞MaaS OpenAI兼容接口
- ReadableStream + TransformStream 流式转发
- 降级策略：无API Key或API错误时回退到页面提示词

### 上下文感知
- 10个页面上下文预置（首页/云游戏/云电脑/免费资源/接码/优惠/搜索/库/网盘/个人中心）
- 每个路由对应独立的系统提示词和推荐tips
- 浏览页面自动触发经验奖励（同路由会话内去重）

### 经验系统
- 聊天+5（日限50）、浏览+3（日限30）、点赞+2（日限20）
- 等级阈值：0/100/300/600/1000
- 升级到Lv.2触发孵化动画
